Overview

Apotheosis Comics is a superhero-themed deckbuilder roguelike made in Unity. The project was built around the idea of creating a fast, replayable combat experience where card choices and status effects matter in every run.

Unity C# Deckbuilder

Features

  • Deckbuilding mechanics and turn-based combat
  • Dynamic status effect interactions
  • Roguelike progression and replayable runs
