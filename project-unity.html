Overview

Apotheosis Comics is a superhero-themed deckbuilder roguelike built in Unity. An unfinished project that explores fast, replayable combat where every decision matters. The core loop combines strategic card choices with a deep status effect system—effects interact with each other and stack in ways that completely change how you approach each run. Gameplay videos are available on the YouTube channel showcasing the combat systems and card interactions in action.

Features

  • Deckbuilding mechanics and turn-based combat
  • Dynamic status effect interactions
  • Roguelike progression and replayable runs
