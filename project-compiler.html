This project is a deeper technical build focused on parsing, language implementation, and understanding how a compiler turns source code into executable behavior.

Overview

The compiler processes a small language subset and translates code through the core stages of tokenization, parsing, and execution. The project was designed to explore how simple programming languages are interpreted and how basic language features can be implemented from scratch.

C Compiler Design Parsing

Implemented Features

  • Tokenization and parsing for a simplified Kotlin-style syntax
  • Support for variables, functions, conditionals, and loops
  • Basic execution flow and dynamic typing support
